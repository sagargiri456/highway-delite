# Highway Delite

A full-stack note-taking app built with React, Node.js, and TypeScript.

## Tech Stack
- Frontend: React + Vite + Tailwind CSS
- Backend: Express + MongoDB
- Auth: Email/OTP + JWT

## Setup
1. Clone repo
2. Run `npm install` in both `frontend` and `backend`
3. Add `.env` files
4. Start dev servers
